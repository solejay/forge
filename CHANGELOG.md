# Changelog

## 0.1.0 - Initial Forge snapshot

### Added

- `@forge/forge-core`
  - world state in `pipeline/state.json`
  - forge status/update/artifact tools
  - drift detection and drift decision tooling
  - worktree delegation
  - reviewer merge gate
  - role-based model routing
  - handoff/context-rot management
  - forge doctor diagnostics
- `@forge/forge-design-studio`
  - design pipeline extension
  - design app, design brief, style guide, microcopy, prompt, QA skills
- `@forge/forge-mobile-dev`
  - mobile routing extension
  - iOS/Android specialist personas
  - mobile loop, project discovery, split, crash, deploy, performance, accessibility, motion skills

### Notes

- This repository is a release monorepo snapshot of the local Forge package family.
