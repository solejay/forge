# Forge v0.1.0

Initial public Forge monorepo release.

## Install

```bash
pi install git:github.com/solejay/forge@v0.1.0
```

or install the latest `main`:

```bash
pi install git:github.com/solejay/forge
```

## Included packages

- `@forge/forge-core`
- `@forge/forge-design-studio`
- `@forge/forge-mobile-dev`

## Highlights

### Forge Core

- `pipeline/state.json` world state
- plan/review skills
- drift detection and explicit drift decisions
- worktree-isolated delegation
- reviewer merge gate
- role-based model routing
- context handoff / context-rot management
- Forge package/install doctor

### Forge Design Studio

- PRD → rendered design artifact pipeline
- design brief generation
- style guide/design token generation
- microcopy generation
- screen prompt generation
- design QA

### Forge Mobile Dev

- iOS/Android/RN routing context
- iOS and Android specialist personas
- mobile visual verification loop
- crash triage
- deployment helpers
- performance, accessibility, code-signing, motion-design skills

## Validation

Release smoke test:

```bash
npm run smoke
```

Validates:

- individual package extension loads
- root monorepo pi manifest loads all Forge resources
- Forge tool discovery

## Notes

- Root package install is supported via `pi install git:github.com/solejay/forge@v0.1.0`.
- Do not install both the root package and individual subpackages at the same time, or duplicate tool registration may occur.
